🚀 API REST com Spring Boot

Este projeto faz parte dos meus estudos utilizando o framework Spring Boot, o framework Java mais utilizado atualmente para desenvolvimento de aplicações backend e APIs REST.

Durante o desenvolvimento deste projeto, são aplicados diversos conceitos importantes do ecossistema Spring para construção de uma API RESTful completa, segura e escalável.

📚 Sobre o Projeto

Neste projeto são abordados conceitos essenciais para desenvolvimento de APIs modernas utilizando Java, incluindo persistência de dados, segurança, validações e boas práticas de arquitetura.

A aplicação utiliza tecnologias como:

Spring Boot

Spring Data JPA

Hibernate

JDBC

Spring Security

JSON Web Token

Project Lombok

🎯 Objetivo

O objetivo deste projeto é aprender como desenvolver uma API REST profissional, utilizando recursos importantes do ecossistema Spring, aplicando boas práticas utilizadas no mercado.

O curso não tem como objetivo cobrir todos os recursos do Spring, mas apresentar conceitos importantes de nível iniciante e intermediário para construção de APIs robustas.

⚙️ Tecnologias utilizadas

Java

Spring Boot

Spring Data JPA

Hibernate

JDBC

Spring Security

JWT

Lombok

Maven

Banco de dados relacional (H2 / MySQL)
